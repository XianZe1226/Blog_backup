# 以下是一些建设过程中遇到的问题：

## 终端显示没有node版本被激活，解决方法：

### 激活 nvm:

在 zsh 里运行：
```bash
source ~/.nvm/nvm.sh
```
然后再运行： 
```bash
nvm list 
```
看看装过的 Node 版本（比如 v20.19.5）还在不在。

### 选择一个 Node 版本：
如果 nvm list 里能看到 v20.19.5，就执行：
```bash
nvm use 20
```

然后再确认：
```bash
node -v
npm -v
```

## 文章封面的默认设置未生效，解决方法：

### 经查发现default_top_img中的路径有误

错误写法：
```Markdown
default_top_img: /source/images/top.jpg
```

正确写法：
```Markdown
default_top_img: /images/top.jpg
```

原因：/images/1.jpg → 指向 source/images/1.jpg，这是 Hexo 的约定，生成时会自动复制到 public/images/1.jpg，所以这样写就够了。

#### ps：YAML语法要求冒号后必须有空格
